## Welcome to Umucraft Launcher

Umucraft Launcher is a custom Minecraft launcher built with Electron that streamlines the modded Minecraft experience for both players and server administrators. It handles the complexity of mod management, Java installation, and game launching through an intuitive desktop application.

## Who is this for?

### Players

Get up and running with modded Minecraft in minutes. The launcher handles Java installation, mod downloads, and game launching automatically. Just enter your username and click play.

### Server Administrators

Distribute mod packs effortlessly. Update your manifest file, and all players automatically receive the latest mods on their next launch. No manual distribution required.

### Developers

Built on a clean, modular architecture with separated concerns. The codebase uses Electron with CommonJS for the main process and ES modules for the renderer, making it easy to extend and customize.

## Technical Overview

Umucraft Launcher is built with:

- **Electron 28** for cross-platform desktop functionality
- **Node.js** for backend services (main process)
- **Vanilla JavaScript** with ES modules for the UI (renderer process)
- **Adoptium JDK** for automatic Java installation
- **Minecraft Forge** installer integration

The launcher manages a complete Minecraft installation at `~/.UmuCraft/`, including Java, game files, mods, assets, and configuration.
